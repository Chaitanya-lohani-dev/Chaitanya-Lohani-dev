I specialize in designing **backend architectures, authentication systems, and production deployments** using modern technologies like **Node.js, Next.js, Docker, Redis, and PostgreSQL/MongoDB**.

# 💡 Key Projects

### 🧑‍💻 Student Internship Portal
A full-stack platform that helps students discover legitimate internships while preventing fake opportunities.

**Highlights**

• Secure authentication with **JWT access + refresh token rotation**  
• **Redis caching** for performance  
• **Dockerized deployment** with multiple services  
• **Nginx reverse proxy architecture**  
• MongoDB database with structured schema  
• Admin dashboard for managing jobs and applications

Tech Stack:  
`Node.js` `Express` `MongoDB` `Redis` `Docker` `Nginx` `Next.js`
